I used linear regression model to predict the amount of time it takes to get somewhere from Uber based on the starting and ending longitude and latitude, using Jupyter Python. By extracting the Euclidean distance between the given longitude and latitude variables and splitting the data to train the model and predict using the test data, I was able to generate an accurate model that predicts the time based off of the Euclidean distance. The folder contains the the dataset, Jupyter notebook file, and the predictions cvs file.
